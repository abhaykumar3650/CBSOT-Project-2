# 🔍 Deep Learning Search Engine

A deep learning-based search engine for retrieving relevant machine learning research papers using NLP techniques.

## 📂 Dataset
- ML ArXiv Papers: https://huggingface.co/datasets/CShorten/ML-ArXiv-Papers

## ⚙️ Features
- Text preprocessing & EDA
- Semantic search (beyond keyword matching)
- Deep learning-based retrieval

## 🛠️ Tech Stack
Python, Pandas, Scikit-learn, NLP, Deep Learning
